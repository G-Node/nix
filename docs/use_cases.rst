NIX in action
=============

We will illustrate the use of NIX in three different contexts:

1) Electrophysiological recordings and intergration of NIX into a recording tool
2) Use of NIX as a storage backend for the `neo <http://neuralensemble.org/neo>`__ data model.
3) Use of NIX to store EEG data.
