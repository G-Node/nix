Detailed Description

Dimension descriptor for a dimension that is irregularly sampled.

The RangeDimension covers cases when indexes of a dimension are mapped to other values in a not regular fashion. A use-case for this would be for example irregularly sampled time-series or certain kinds of histograms. To achieve the mapping of the indexes an array of mapping values must be provided. Those values are stored in the dimensions ticks property. In analogy to the sampled dimension a unit and a label can be defined.
